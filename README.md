## 1. Introduction

Python is one of the most versatile and beginner-friendly programming languages used in software development, data science, web applications, and artificial intelligence.  
This paper focuses on five important Python concepts:

1. **Array Methods**  
2. **String Methods**
3. **Dictionaries Methods**
4. **Objects and Object-Oriented Programming (OOP)**
5. **SOLID Principles**

Each section explores syntax, commonly used methods, and examples to demonstrate practical usage.
